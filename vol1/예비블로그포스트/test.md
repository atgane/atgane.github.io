# 0. 소개

텐서를 알아보기 전에 벡터를 복습해보자. 벡터를 만약 벡터공간의 원소로 알고 있고 극좌표계의 좌표 $(r, \theta)$가 벡터가 안되는 이유를 알고 있다면 굳이 안읽어도 된다. 개인적으로 처음에 극좌표가 벡터가 안되는 이유조차 몰라서 굉장히 헤맸던 기억이 있어 관련해서 글을 쓰게 되었다.
